# Nodebird

This project clones Twitter application using express.js



## Features

- Authentication
    - Sign up
    - Log in
    - Log out
- Post
    - CRUD text
    - Upload image files
    - Categorize by hashtags
- Network
    - Follow other users
    - Show followers



## Tech Stacks

- Node.js / Express.js
- MySQL / Sequelize
- Passport / kakao login API
